# 黄钺：七日粮（HuangYue: Seven Days of Grain）

> *"盗跖庄屩流誉后，更陈王奋起挥黄钺。" —— 毛泽东《贺新郎·读史》*
>
> *黄钺从来不是帝王之器。它是被逼到绝路的人，拿起的最后一件东西。*

---

## 这不是一个英雄叙事

大明崇祯十四年，河南。你带着二百八十七名饥民与义军来到临河县。军中存粮只够三日。

市面上大多数策略游戏让你扮演国家意志、星际霸主、百年王朝。**黄钺让你扮演一个带着饥民求活的人。**

你不是刘邦，不是朱元璋。你只是一个在乱世里要做选择的人——而每一个选择，都有人替你承担代价。

---

## 跟别的策略游戏有什么不一样

### 民心不是参数，是镜子

传统策略游戏：民心 +5 / -10，影响产出效率。

**黄钺**：没有数值告诉你"你是义军还是匪"。你做的事，由百姓的反应告诉你——

- 你是义军：村民开门，小孩跟着队伍跑，有人在路边放水罐
- 你是匪：进村之前钟声已响，青壮年全上山，井水有股怪味
- 灰色地带：村民白天关门，夜里有人偷偷送粮出来

**你自己可能最后一个知道，你变成了什么。**

### 赢了也不是英雄

胜利不是你打下了一座城。胜利是你的辎重队过了河，有几十个百姓活过了这个冬天。史书不会写你们的名字——但赵先生的薄册上记着每一个死者。

败了也不是国破家亡。败了是粮车旁的人没留下姓名，官军簿子上只写"击溃流贼"。

## 当前版本 v0.4

### 七日文字模式
- 七个连续事件，每日两项选择
- 余粮、部众、军心、民望、疫病五项状态
- 隐藏的"残酷"倾向（你不看攻略不会知道它的存在）
- 四类动态结局
- 陈二娘、石头、赵先生、韩九、周禾生五人的动态小传——每个人的结局文字根据你的选择生成

### 实时战场模式（新）
- 水墨军图风格地图
- 主力营、游骑哨、辎重队三支可调动部队
- Dijkstra 道路寻路、平滑起停加减速
- 主力与游骑从不同方向接敌形成**夹击**
- 设伏阻击：扩大拦截范围 + 首次伏击伤害
- 目标：击败仓守 → 辎重队装粮 → 撤往南渡口
- 民望实时影响：行军速度、战力、是否有人带路
- 程序绘制插画的动态突发事件（粮道被封、流民来投、暴雪行军、军中谣言……）

## v0.5 设计方向（欢迎讨论）

1. **民望去数值化**：不再显示"民望：52"，用村民的行为反映你的道德位置
2. **镜像反馈**：你的行为 → 村民反应 → 你从他们的反应里看见自己变成了什么
3. **更多非战斗困境**：断粮、疫病、内部分裂——战争不只是在战场上输的
4. **没有"正确"选择**：每个选项都有代价，区别只是谁来付

## 运行方法

1. 下载最新 Release 中的 ZIP 文件
2. 解压到任意目录
3. 双击 `黄钺-七日粮-Demo.exe`
4. 如果 Windows 提示"未知发布者"，点击"更多信息 → 仍要运行"

> 本游戏是个人制作原型，尚未购买商业数字签名证书。请确认文件来自本仓库的 Release 页面。

**系统要求**：Windows 10 / 11，无需安装任何运行库。

## 技术栈

- C# (.NET Framework)
- Windows Forms（纯 WinForms，无第三方游戏引擎）
- 程序生成所有插画（六个场景类型：村庄、官仓、流民、伤兵、谣言、暴雪）
- 水墨风格地图实时渲染
