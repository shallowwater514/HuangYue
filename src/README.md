# 黄钺：七日粮 Demo

一个以明末河南为背景的个人游戏原型。v0.4 同时包含实时战场和原有的七日叙事模式。

## 当前内容

- 七个连续事件，每日两项选择
- 余粮、部众、军心、民望、疫病五项状态
- 隐藏的“残酷”倾向
- 四类动态结局
- 陈二娘、石头、赵先生、韩九、周禾生五人的动态小传
- 原生 Windows 图形界面，无需安装额外运行库
- 水墨军图风格的实时地图
- 主力营、游骑哨、辎重队三支可调动部队
- 沿道路节点寻路、平滑起停、自动交战、暂停与两档速度
- 主力营与游骑哨从不同方向接敌时形成夹击
- 主力营或游骑哨可就地设伏，扩大拦截范围并发动首次伏击
- 夺取官仓、装载粮食并撤往南渡口的实时目标
- 根据粮秣、伤亡、军心和民望抽取的动态突发事件
- 民望低迷、崩溃与拥护三类实时行军和战斗效果
- 带程序绘制插画的战场事件窗口

## 构建

在 PowerShell 中运行：

```powershell
.\build.ps1
```

生成文件位于 `dist\HuangYueDemo.exe`。复制成品到目标位置时，可以将它重命名为中文名称。

## 操作

双击 EXE，点击“举起黄钺”，每天从两个方案中选择一个。第七次选择后进入“乱世录”。

> 本 Demo 的县名、人物与具体事件均为虚构，时代氛围取自明末灾荒与战争背景。
